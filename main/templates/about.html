About the Titanic Dataset

Understanding the historical data, methodology, and machine learning approach behind survival predictions

Dataset Information

Real passenger data from the 1912 Titanic disaster

Dataset Details

  • Training Samples: 891 passengers
  • Test Samples: 418 passengers
  • Features: 11 attributes
  • Target: Survived (0 = No, 1 = Yes)
  • Source: Kaggle Competition

Key Features

  • Pclass: Passenger class (1st, 2nd, 3rd)
  • Sex: Gender of passenger
  • Age: Age in years
  • SibSp: # of siblings/spouses aboard
  • Parch: # of parents/children aboard
  • Fare: Passenger fare paid
  • Embarked: Port of embarkation

ML Methodology

Perceptron algorithm for binary classification

Algorithm: Perceptron

The Perceptron is a simple neural network algorithm for supervised learning of binary classifiers. It learns a linear decision boundary by iteratively adjusting weights based on misclassified examples. The algorithm predicts survival (1) or non-survival (0) based on passenger features.

ŷ = activation(w₀ + w₁x₁ + w₂x₂ + ... + wₙxₙ)
where activation(z) = 1 if z ≥ 0, else 0

Training Phase

Weights are updated iteratively using the perceptron learning rule on training data

Weight Updates

w = w + η(y - ŷ)x where η is the learning rate

Prediction

New passengers classified using learned decision boundary

Feature Descriptions

Pclass (Passenger Class)

A proxy for socio-economic status. 1st class passengers had higher survival rates.

Values: 1, 2, 3

Sex (Gender)

"Women and children first" policy significantly affected survival rates.

Values: male, female

Age

Children had higher priority for lifeboats. Age ranges from 0.42 to 80 years.

Continuous variable

SibSp (Siblings/Spouses)

Number of siblings or spouse aboard. Family size affected survival probability.

Values: 0-8

Parch (Parents/Children)

Number of parents or children aboard. Related to family group dynamics.

Values: 0-6

Fare

Ticket price paid. Correlates with class and cabin location on the ship.

Range: £0 - £512

Embarked (Port)

Port of embarkation: C=Cherbourg, Q=Queenstown, S=Southampton.

Values: C, Q, S

Historical Insight

The "women and children first" protocol, combined with socio-economic factors (passenger class), were the strongest predictors of survival. First-class passengers had better access to lifeboats, while third-class passengers faced locked gates and greater distances to the deck.
